---

## 🚀 Cómo correr el proyecto localmente

1. Copia la carpeta `estacion-arcade/` dentro de `C:/xampp/htdocs/`
2. Inicia Apache desde el panel de XAMPP
3. Abre tu navegador en: `http://localhost/estacion-arcade/`
